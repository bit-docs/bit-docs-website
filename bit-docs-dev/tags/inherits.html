@inherits module

Indicates that the current DocObject inherits from another DocObject.

@inherits NAME 

/*
 * @constructor Client
 * @inherits Person
 * ...
 */
 var Client = Person.extend({})

Parameters

  1. NAME {String}:

    The name of DocObject the current DocObject inherits from.
