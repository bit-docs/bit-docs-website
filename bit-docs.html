bit-docs module

This package provides the command-line interface and and plugin framework for bit-docs.

bitDocs(pathToPackageJSON, options)

Bootstraps a command-line interface and plugin framework that provides the necessary scaffolding for an ecosystem of plugins that find files to read in as FileSource, process FileSource to make a DocMap from DocObjects, or take a DocMap and generate a static distributable.
