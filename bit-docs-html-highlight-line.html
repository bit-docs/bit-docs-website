bit-docs-html-highlight-line module

Adds a @highlight tag. Needs to be used after bit-docs-prettify.

function()

Returns

{undefined}:

This plugin registers onto these hooks:

  • tags
  • html

Registering the tags hook adds the @highlight tag.

Registering the html hook adds a static JavaScript file used to do the highlighting bit-docs-html-highlight-line/highlight-line.js.
