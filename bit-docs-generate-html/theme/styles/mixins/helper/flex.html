.flex

Sets flexbox "grow" and "shrink" in most compatible way.

#helper.flex(@grow: 0, @shrink: 1)

Parameters

  1. @grow {Number}:

    The amount this flex item should grow, defaults to 0.

  2. @shrink {Number}:

    The amount this flex item should shrink, defaults to 1.

Use

.flex-none {
  #helper.flex(0; 0);
}
