@parent

Specifies the parent [documentjs.process.docObject]'s name. The current docObject will be displayed under the the parent in the navigation.

@parent NAME 

/**
 * @constructor jQuery.Drag
 * @parent specialevents
 * ...
 */

Parameters

  1. NAME {String}:

    The name of the parent [documentjs.process.docObject].

Use

[documentjs.tags.function] and [documentjs.tags.property] tags can infer their parent from the current scope.
