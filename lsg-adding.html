Adding to Existing JS Docs page

This guide will:

  • Give a designer-friendly explanation of what DocumentJS does
  • Make sure you have everything you need installed
  • Help you configure DocumentJS to add a Live Style Guide
  • Explain how to use tags to write your Live Style Guide

You should start elsewhere if:

  • You want to create a Live Style Guide on a project that doesn't already use DocumentJS
  • You still need to set up DocumentJS for API documentation
  • You're just trying to kill time
