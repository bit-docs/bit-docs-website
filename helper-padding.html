#helper.padding property

Sets padding for passed sides.

#helper.padding(@sides; @amount: @defaultPadding);

Parameters

  1. @sides {String}:

    A space-separated list of "sides", like left right.

  2. @amount {Number}:

    An amount of padding in pixels, defaults to @defaultPadding.

Use

.container {
  #helper.padding(left right; 50px);
}
