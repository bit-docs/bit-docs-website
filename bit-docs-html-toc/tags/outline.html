@outline module

Controls the outline behavior.

@outline DEPTH TAG

Controls the outline's depth and type.

Parameters

  1. DEPTH {Number}:

    The number of headers below and including h2. For example, 2 will include h2 and h3 elements.

  2. TAG {String}:

    Make the list an ordered list.
