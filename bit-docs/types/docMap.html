DocMap typedef

An object that contains every DocObject keyed by the DocObject's name.

Object<String,DocObject>

Example of a docMap:

{
  "api": {
    "src": {
      "path": "docs/api.md"
    },
    "body": "...",
    "description": "...",
    "name": "api",
    "title": "API",
    "type": "page",
    "parent": "homepage",
    "order": 1
  },
  "my-module": {
    "name": "my-module",
    "type": "module",
    "parent": "api",
    "src": {

    },

      {
