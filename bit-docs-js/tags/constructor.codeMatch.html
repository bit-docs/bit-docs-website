codeMatch function

codeMatch(codeLine)

CodeTag.codeMatch for @constructor.

Parameters

  1. codeLine {String}:

    String of characters until newline.

    For example:

    jQuery.extend( "myObj", {

    Will be matched against this regex:

    /([\w\.\$]+?).extend\(\s*["']([^"']*)["']/ // code match

Returns

{Boolean}:

true if string matched regex, false if not.
