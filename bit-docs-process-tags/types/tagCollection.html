TagCollection typedef

A collection of tags.

Object<String,Tag>

Use

Will look like:

{
    author: {
        add: [Function: add],
        name: 'author'
    },
    demo: {
        add: [Function: add]
    },
    // ...more tags added by plugins
    body: {
        add: [Function: add],
        done: [Function: done],
        name: 'body'
    },
    description: {
        add: [Function: add],
        name: 'description'
    },
    hide: {
        add: [Function: add],
        name: 'hide'
    },
    parent: {
        add: [Function: add],
        name: 'parent'
    }
}
