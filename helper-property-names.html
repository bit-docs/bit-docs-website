#helper.property-names property

Helper for setting multiple dash properties to same value.

#helper.property-names(@p; @s; @v: 0);

Parameters

  1. @p {String}:

    The property name to prepend before the dash and string.

  2. @s {String}:

    A space-separated list of strings to iterate over.

  3. @v {Number}:

    The value to set each concatenated property to, defaults to 0.

Use

.top-left-margin {
  .property-names(margin; top left; 100px);
}

.special img {
  .top-left-margin;
}
