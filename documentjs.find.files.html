documentjs.find.files

.find.files(siteConfig)

Parameters

  1. siteConfig {Object}:

    Options that configure the behavior of the files that will be processed.

    • glob {String|documentjs.find.globObject}

      The glob option either specifies a minmatch pattern like:

      documentjs.find.files({glob: "*.js"})

      Or a [documentjs.find.globObject GlobObject] that specifies the a minmatch pattern and other siteConfig like:

      documentjs.find.files({
  glob: {
    pattern: "*.js",
    cwd: __dirname
  }
})

Returns

{documentjs.process.types.FileEventEmitter}:

An event emitter that emits events for matched files.
