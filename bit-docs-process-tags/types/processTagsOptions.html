ProcessTagsOptions typedef

Parameter type expected by bit-docs-process-tags/process-tags.

Object

Options

  • tags {TagCollection}:

    The TagCollection to draw from when processing the TagBlock.

  • block {TagBlock}:

    The TagBlock to be processed.

  • scope {DocObject}:

    A DocObject that can be a parent to the current DocObject.

  • docMap {DocMap}:

    The map of all DocObjects.

  • docObject {DocObject}:

    If provided, this will be used as the DocObject.

    This is useful for adding properties to an existing object.
